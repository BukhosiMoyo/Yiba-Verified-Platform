\## 1. Purpose of This Document

This document lists \*\*all application screens\*\* required for Version
1 (V1) of the Yiba Wise Compliance & QCTO Oversight App.

It acts as:

\- A \*\*single source of truth\*\* for UI/UX design

\- A \*\*development checklist\*\*

\- A guardrail against feature creep

No screen should be built unless it appears in this document or is added
via a version update.
